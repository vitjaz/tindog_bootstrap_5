# Tindog

Одностраничный сайт знакомств для собак по мотивам [Tinder](https://tinder.com/). 

Основная концепция - яркий дизайн и адаптивность к любым размерам экрана на любых устройствах.


## Технологии
+ HTML/SCSS
+ Bootstrap 5
